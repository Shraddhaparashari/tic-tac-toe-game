# Tic Tac Toe Game 

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

- Two player game (X & O)
- Winner detection
- Reset & New Game button
- Clean and simple UI

## Technologies Used

- CSS
- JavaScript (Vanilla JS)
